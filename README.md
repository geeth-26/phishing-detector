# 🔐 Phishing Website Detection System

## 📌 Overview
A web-based application that detects phishing websites using Machine Learning and rule-based techniques.

## 🚀 Features
- Detect phishing URLs
- Machine Learning model (Random Forest)
- Rule-based detection for accuracy
- Confidence score display
- User-friendly web interface

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- HTML, CSS

## 📊 Dataset
- UCI Phishing Websites Dataset

## ⚙️ How to Run
```bash
pip install -r requirements.txt
python app.py
